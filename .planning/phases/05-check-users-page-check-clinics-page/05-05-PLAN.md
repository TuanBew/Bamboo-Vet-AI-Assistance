**`getCheckClinicsData` implementation:**
1. `const db = createServiceClient()`
2. Query `clinics` table for all rows. Apply `.ilike('name', '%search%')` if search set. Apply `.eq('type', clinic_type)` if filter set. Apply `.eq('province', province)` if filter set. Get total count.
3. Query `profiles` where `is_admin = false` to get user_id -> clinic_id mapping.
4. Query `mv_monthly_queries` for the specified `year` to get user-level monthly data.
5. Aggregate: for each clinic, sum query_count (or session_count based on metric) across all users belonging to that clinic (profiles.clinic_id = clinic.id). Build `monthly_data: { "1": total, "2": total, ..., "12": total }`.
6. Derive region/zone from province:
   ```typescript
   function getRegionZone(province: string): { region: string; zone: string } {
     const north = ['Ha Noi', 'Hai Phong', 'Thai Nguyen']
     const central = ['Da Nang', 'Thua Thien Hue', 'Khanh Hoa']
     const south = ['TP. Ho Chi Minh', 'Can Tho', 'Dong Nai', 'Binh Duong']
     if (north.includes(province)) return { region: 'Mien Bac', zone: 'Dong Bang Song Hong' }
     if (central.includes(province)) return { region: 'Mien Trung', zone: 'Duyen Hai Mien Trung' }
     if (south.includes(province)) return { region: 'Mien Nam', zone: 'Dong Nam Bo' }
     return { region: 'Khac', zone: 'Khac' }
   }
   ```
7. Apply pagination (slice in JS since dataset is small — 12 clinics).
8. Return paginated result.

**`getClinicDetail` implementation:**
1. Query `clinics` where `code = facilityCode` to get clinic name and id.
2. Query `profiles` where `clinic_id = clinic.id AND is_admin = false` to get all staff at this clinic.
3. Query `mv_daily_queries` where `year = year AND month = month` and `user_id IN (staff user_ids)`.
4. For each staff user, build `daily_data` array: for each day 1..daysInMonth(year, month), look up the query_count and session_count from mv_daily_queries. If no data for that day, use 0 for both.
5. Return ClinicDetailData with staff_code = profile.id.substring(0, 8).

import { NextRequest, NextResponse } from 'next/server'
import { requireAdmin } from '@/lib/admin/auth'
import { getCheckClinicsData } from '@/lib/admin/services/check-clinics'

export async function GET(request: NextRequest) {
  const auth = await requireAdmin()
  if (auth instanceof NextResponse) return auth

  const { searchParams } = new URL(request.url)
  const year = parseInt(searchParams.get('year') || String(new Date().getFullYear()))
  const metric = searchParams.get('metric') || 'query_count'
  const clinic_type = searchParams.get('clinic_type') || ''
  const province = searchParams.get('province') || ''
  const search = searchParams.get('search') || ''
  const page = parseInt(searchParams.get('page') || '1')
  const page_size = parseInt(searchParams.get('page_size') || '10')

  try {
    const data = await getCheckClinicsData({ year, metric, clinic_type, province, search, page, page_size })
    return NextResponse.json(data)
  } catch (error) {
    console.error('Check clinics API error:', error)
    return NextResponse.json({ error: 'Internal server error' }, { status: 500 })
  }
}

  const body = paginatedRows.map(row => [
