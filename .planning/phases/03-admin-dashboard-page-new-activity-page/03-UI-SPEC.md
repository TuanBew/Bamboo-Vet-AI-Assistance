## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline element padding, chart legend spacing |
| sm | 8px | Compact element spacing, filter bar inner gaps |
| md | 16px | Default card padding, chart container padding, table cell padding |
| lg | 24px | Section padding inside SectionHeader, gap between KPI card rows |
| xl | 32px | Gap between major dashboard sections |
| 2xl | 48px | Vertical space between "Tong quan" and "Chi so tap trung" sections |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Filter bar height: 52px (non-standard, accommodates select + search input comfortably)
- KPI card minimum height: 100px (ensures icon + value + label stack without cramping)
- Map container height: 400px (fixed, sufficient for Vietnam country view at zoom 6)
- Sparkline cell: 120px width x 30px height (inline in table rows)
- Inline stacked BarChart cell: 120px width x 24px height (inline in table rows)

## Chart Specifications

### Dashboard — "Tong quan" Section

| Chart | Type | Height | Data |
|-------|------|--------|------|
| Yearly totals | Recharts `<BarChart>` grouped | 280px | 2024/2025/2026 bars for queries + sessions |
| Monthly trend + forecast | Recharts `<ComposedChart>` with `<Area>` + `<Line>` | 280px | Jan 2024 to current month (solid area) + 3 forecast months (dotted `<Line>` with `strokeDasharray="4 4"`) |

**Forecast line:** Render as a separate `<Line>` series overlaying the `<Area>`. Only `is_forecast: true` data points are plotted on this line. Use same cyan color with `strokeDasharray="4 4"` and `strokeOpacity={0.7}`.

### Dashboard — "Chi so tap trung" Section

| Chart | Type | Height | Data |
|-------|------|--------|------|
| Daily volume | Recharts `<LineChart>` | 240px | Day 1-31 of selected month, query_count |
| 6 Donut charts | Recharts `<PieChart>` | 200px each | 3 for queries (drug_group, animal_type, query_type) + 3 for sessions |

**Donut layout:** 2 rows of 3 donuts. Row 1 labeled "Ti trong nhap hang thang MM-YYYY" (queries). Row 2 labeled "Ti trong ban hang thang MM-YYYY" (sessions). Each donut has a center label showing the category name.

### Dashboard — "Nguoi dung" Section

| Element | Type | Height | Data |
|---------|------|--------|------|
| User leaderboard table | HTML table (not DataTable) | Auto (20 rows max) | Top 20 users sorted by total_queries desc |
| Drug group breakdown | Recharts `<BarChart>` horizontal stacked, inline | 24px per row | Per-user drug group distribution |
| Query type breakdown | Recharts `<BarChart>` horizontal stacked, inline | 24px per row | Per-user query type distribution |
| 12-month sparkline | `<SparklineChart>` (existing component) | 30px | monthly_sparkline array from API |

**Table columns:** Ten nhan vien | Doanh so theo ngay (sparkline) | TOTAL | Don hang (sessions) | Trung binh | Khach hang (queries) | Ngay > 1tv (days active)

### Dashboard — "Phong kham" Section

## Interaction States

### Filter Bar Behavior

| Interaction | Behavior |
|-------------|----------|
| Filter change | Immediate refetch (no submit button, no debounce). URL searchParams updated. |
| Loading during refetch | Full section skeleton replaces chart/table area. KPI cards remain visible (they are unfiltered). |
| Filter reset | Clearing a select returns to "all" (null). Charts reload with unfiltered data. |

### Chart Interactions

| Interaction | Behavior |
|-------------|----------|
| Hover on chart data point | Recharts Tooltip appears: dark bg, white text, value formatted with locale separator |
| Hover on PieChart slice | Slice expands slightly (Recharts `activeShape`), tooltip shows category name + count + percentage |
| Click on map pin | Leaflet popup: clinic name + total query count. Popup auto-closes when another pin is clicked. |
| Chart legend click | Recharts default: toggles series visibility |
| SectionHeader collapse | Click chevron toggles section visibility. Default: all sections open. |

### Table Interactions (Dashboard user table)

| Interaction | Behavior |
|-------------|----------|
| Row hover | `bg-gray-700/50` highlight |
| No click action | Leaderboard is read-only in Phase 3 (no drill-down) |
| Sorting | Not applicable — always sorted by total_queries desc |

### Skeleton Loading Pattern

When filters change and data is refetching:
- KPI cards: remain visible (unfiltered data, no skeleton)
- Charts: replaced with `bg-gray-700 animate-pulse rounded-lg` blocks at declared chart heights
- Tables: replaced with 5 rows of `bg-gray-700 animate-pulse rounded h-8` bars
- Map: overlay with semi-transparent `bg-gray-900/50` + centered spinner
